## Design System

### Color Palette

**Primary Colors:**
- Primary: `#3b82f6` (Blue) - Main actions, links
- Secondary: `#64748b` (Slate) - Secondary text, borders
- Accent: `#8b5cf6` (Purple) - Highlights, special features

**Status Colors:**
| Status | Color | Hex |
|--------|-------|-----|
| Pending | Gray | `#6b7280` |
| Assigned | Blue | `#3b82f6` |
| On the Way | Cyan | `#06b6d4` |
| Met Customer | Teal | `#14b8a6` |
| Order Completed | Lime | `#84cc16` |
| Docket Received | Green | `#22c55e` |
| Docket Uploaded | Emerald | `#10b981` |
| Ready to Invoice | Indigo | `#6366f1` |
| Invoiced | Violet | `#8b5cf6` |
| Completed | Green | `#16a34a` |
| Customer Issue | Orange | `#f97316` |
| Building Issue | Yellow | `#eab308` |
| Network Issue | Pink | `#ec4899` |
| Rescheduled | Purple | `#a855f7` |
| Withdrawn | Red | `#ef4444` |

**Neutral Colors:**
- Background: `#ffffff` (White)
- Surface: `#f8fafc` (Light Gray)
- Border: `#e2e8f0` (Gray 200)
- Text Primary: `#0f172a` (Slate 900)
- Text Secondary: `#64748b` (Slate 500)

### Typography

**Font Family:** Inter (sans-serif)

**Font Sizes:**
- Heading 1: 30px / 1.875rem (bold)
- Heading 2: 24px / 1.5rem (semibold)
- Heading 3: 20px / 1.25rem (semibold)
- Body: 16px / 1rem (regular)
- Small: 14px / 0.875rem (regular)
- Tiny: 12px / 0.75rem (regular)

### Spacing

**Base Unit:** 4px

**Common Spacings:**
- xs: 4px
- sm: 8px
- md: 16px
- lg: 24px
- xl: 32px
- 2xl: 48px

### Components

**Buttons:**
- Primary: Blue background, white text
- Secondary: White background, blue border
- Destructive: Red background, white text
- Ghost: Transparent, hover gray

**Inputs:**
- Border: 1px solid gray-300
- Focus: Blue ring
- Error: Red border + red text
- Disabled: Gray background

**Cards:**
- Background: White
- Border: 1px solid gray-200
- Shadow: sm (0 1px 2px rgba(0,0,0,0.05))
- Radius: 8px

## Responsive Behavior

### Desktop (> 1024px)
- Full table with all columns visible
- Sidebar navigation
- Multi-column schedule view
- Drag-and-drop enabled

### Tablet (640px - 1024px)
- Scrollable table (horizontal scroll)
- Collapsed sidebar (hamburger menu)
- 2-column schedule view
- Touch-optimized buttons

### Mobile (< 640px)
- Card-based list view (no table)
- Bottom navigation
- Single-column schedule
- Stacked forms
- No drag-and-drop

---

## Interaction Patterns

### Loading States
- Skeleton screens for tables
- Spinner for buttons during actions
- Progress bar for file uploads

### Empty States
- "No orders found" with illustration
- "Add your first installer" CTA
- "No assignments for this date" message

### Error States
- Toast notifications for errors
- Inline validation messages
- Retry buttons for failed actions

### Success States
- Toast notifications for success
- Checkmark animations
- Auto-dismiss after 3 seconds
